Determine the sample statistics (x‾\bar{x} and ss). I would recommend using a spreadsheet or R (or another computer-based method). x‾=55.42\bar{x} = 55.42 s=66.49s = 66.49 Evaluate the tt statistic.

t=|55.42−0|(66.49)/12=2.89t = \frac{|55.42-0|}{(66.49)/\sqrt{12}} = 2.89 Restate the pp-value. p-value=P(|T|>2.89)p\text{-value} = P\left(|T|>2.89\right) Remember, the degree of freedom is one less than the sample size. df=11\text{df} = 11

At this point, there are various ways to determine the pp-value. The least accurate way is to use the tt-table. Go to the row with df=n−1=12−1=11\text{df}=n-1=12-1=11.


If we go to the row with df=11, we can see that our calculated tt, 2.89, is between 2.72 and 3.11. Thus, we know P(|T|>2.89)P(|T|>2.89) is between 0.01 and 0.02. You could get pretty close by using linear interpolation. P(|T|>2.89)≈(2.89−2.723.11−2.72)(0.01−0.02)+0.02=0.0156P(|T|>2.89)\approx \left(\frac{2.89-2.72}{3.11-2.72}\right)\left(0.01-0.02\right)+0.02 = 0.0156

The more accurate pp-value is 0.0147823. This can be calculated using a computer; for example, you can use this web app (for full accuracy, you’ll need the more precise value of t=2.8871025t=2.8871025). You can also use a spreadsheet or R.
