Determine the sample statistics (x‾\bar{x} and ss). I would recommend using a spreadsheet or R (or another computer-based method). x‾=7.65\bar{x} = 7.65 s=38.46s = 38.46 Evaluate the tt statistic.

t=|7.65−0|(38.46)/60=1.54t = \frac{|7.65-0|}{(38.46)/\sqrt{60}} = 1.54 Restate the pp-value. p-value=P(|T|>1.54)p\text{-value} = P\left(|T|>1.54\right) Remember, the degree of freedom is one less than the sample size. df=59\text{df} = 59

At this point, there are various ways to determine the pp-value. The least accurate way is to use the tt-table. Go to the row with df=n−1=60−1=59\text{df}=n-1=60-1=59.


If we go to the row with df=59, we can see that our calculated tt, 1.54, is between 1.3 and 1.67. Thus, we know P(|T|>1.54)P(|T|>1.54) is between 0.1 and 0.2. You could get pretty close by using linear interpolation. P(|T|>1.54)≈(1.54−1.31.67−1.3)(0.1−0.2)+0.2=0.135P(|T|>1.54)\approx \left(\frac{1.54-1.3}{1.67-1.3}\right)\left(0.1-0.2\right)+0.2 = 0.135

The more accurate pp-value is 0.1287035. This can be calculated using a computer; for example, you can use this web app (for full accuracy, you’ll need the more precise value of t=1.5408357t=1.5408357). You can also use a spreadsheet or R.
