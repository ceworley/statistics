First, draw a sketch. We can label the HH axis by adding integer multiples of 1 to 18. We know to shade toward the right because large values of HH satisfy the condition H>17H>17.


We are given a specific hh value as a boundary. (Remember, for random variables we use uppercase letters, but for specific values we use lowercase.) h=17h = 17 We calculate the zz value of the boundary. z=h−μσ=17−181=−1\begin{aligned} z &= \frac{h-\mu}{\sigma} \\\\ &= \frac{17-18}{1} \\\\ &= -1 \end{aligned}

We have rephrased our problem into a standard normal probability problem, because P(H>17)=P(Z>−1)P(H>17) ~=~ P(Z>-1)

So, we just need to evaluate P(Z>−1)P(Z>-1).

To do this, you need to remember that right-area events are complementary to left-area events. P(Z>−1)=1−P(Z<−1)P(Z>-1) ~=~ 1-P(Z<-1)

You can use a zz-table.
