First, determine the interval of typical measurements.

interval of typical measurements=(μ−2σ,μ+2σ)=(50,90)\begin{aligned} \text{interval of typical measurements} &= (\mu-2\sigma,\,\mu+2\sigma)\\ &= (50,90) \end{aligned}

Now, determine how many measurements (and divide by 200 for what proportion) are either less than 50 or more than 90. You’ll want to use a computer.

R

x = read.csv("check_interval_typical_measurements.csv")$x
n = length(x)
count_outside = sum(x<50 | x>90)
prop_outside = count_outside/n
print(prop_outside)
## [1] 0.035

In R, the “|” operator means “or”.

You could also write the equality as an absolute deviation from the mean. Any measurement more than 20 units from 70 (in either direction) would be outside the interval.

x = read.csv("check_interval_typical_measurements.csv")$x
n = length(x)
count_outside = sum( abs(x-70)>20 )
prop_outside = count_outside/n
print(prop_outside)
## [1] 0.035

Spreadsheet

In a spreadsheet you can use the IF function along with the OR function to determine which measurements are under 50 or over 90. You then use the SUM function to count the 1s.
