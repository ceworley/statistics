First, determine \(z^\star\) such that \(P(|Z|<z^\star)=0.91\). It can help to draw a sketch.


The entire area under the density curve is 1. Thus, we can determine the areas of the tails (using symmetry). \[\text{area of each tail} = \frac{1-\gamma}{2} = \frac{1-0.91}{2} = 0.045\]


Determine a leftward area with boundary \(z^\star\). \[P(Z<z^\star)=\gamma + \frac{1-\gamma}{2}= 0.91+0.045=0.955\] It should be mentioned the expression can be simplified. \[P(Z<z^\star)= \frac{1}{2}+\frac{\gamma}{2}= 0.5+0.455=0.955\]


We rephrase the puzzle. We wish to determine \(z^\star\) such that \(P(Z<z^\star)=0.955\). This is easy to determine using a \(z\)-table, qnorm in R, norm.inv in spreadsheets, or other methods for evaluating the quantile function.

\[P(Z<1.7)=0.955\] In the original phrasing,

\[P(|Z|<1.7)=0.91\]

So, \(z^\star = 1.7\). Now, use the given expressions.

\[\text{Lower bound} = 35.7-(1.7)(10.6)/\sqrt{278} ~=~ 34.6\] \[\text{Upper bound} = 35.7+(1.7)(10.6)/\sqrt{278} ~=~ 36.8\]

Spreadsheet

A spreadsheet can use the confidence.norm function. It takes three arguments:

  • Alpha (\(\alpha\)) is the two-tail area, \(\alpha=1-\gamma=1-0.91=0.09\)
  • Population standard deviation, \(\sigma=10.6\)
  • Sample size, \(n=278\)

The function returns the margin of error. \[\text{margin of error} = \frac{z^\star \sigma}{\sqrt{n}}\]

So, in this case, if in a spreadsheet you typed =confidence.norm(1-0.91,10.6,278), the result would be the margin of error, 1.0778419. You then add/subtract the margin of error to/from the sample mean. \[\bar{x}-\text{ME} = 35.7-1.078 = 34.622\] \[\bar{x}+\text{ME} = 35.7+1.078 = 36.778\]

You can do this with a spreadsheet.


R

xbar = 35.7
sigma = 10.6
n = 278
gamma = 0.91
zstar = qnorm(gamma+(1-gamma)/2)
ME = zstar*sigma/sqrt(n)
LB = xbar-ME
UB = xbar+ME
cat(sprintf("The lower bound: %.4f\nThe upper bound: %.4f",LB,UB))
