First, determine \(t^\star\) such that \(P(|T|<t^\star)=0.8\). It can help to draw a sketch.


The entire area under the density curve is 1. Thus, we can determine the areas of the tails (using symmetry). \[\text{area of each tail} = \frac{1-\gamma}{2} = \frac{1-0.8}{2} = 0.1\]


Thus, we determine a left area. \[P(T<t^\star)=\gamma + \frac{1-\gamma}{2}= 0.8+0.1=0.9\] It should be mentioned the expression can be simplified. \[P(T<t^\star)= \frac{1}{2}+\frac{\gamma}{2}= 0.5+0.4=0.9\]


We rephrase the puzzle. We wish to determine \(t^\star\) such that \(P(T<t^\star)=0.9\). This is easy to determine using a \(t\)-table, qt in R, t.inv in spreadsheets, or other methods for evaluating the quantile function.

Because \(n=34\), then \(\text{df}=33\).

\[P(T<1.31)=0.9\] In the original phrasing,

\[P(|T|<1.31)=0.8\]

So, \(t^\star = 1.31\). Now, use the given expressions.

\[\text{Lower bound} = 87.8-(1.31)(17.7)/\sqrt{34} ~=~ 83.8\] \[\text{Upper bound} = 87.8+(1.31)(17.7)/\sqrt{34} ~=~ 91.8\]

Spreadsheet

A spreadsheet can use the confidence.t function. It takes three arguments:

  • Alpha (\(\alpha\)) is the two-tail area, \(\alpha=1-\gamma=1-0.8=0.2\)
  • sample standard deviation, \(s=17.7\)
  • Sample size, \(n=34\)

The function returns the margin of error. \[\text{margin of error} = \frac{t^\star s}{\sqrt{n}}\]

So, in this case, if in a spreadsheet you typed =confidence.t(1-0.8,17.7,34), the result would be the margin of error, 3.9696687. You then add/subtract the margin of error to/from the sample mean. \[\bar{x}-\text{ME} = 87.8-3.97 = 83.83\] \[\bar{x}+\text{ME} = 87.8+3.97 = 91.77\]

You can do this with a spreadsheet.


R

xbar = 87.8
s = 17.7
n = 34
gamma = 0.8
df = n-1   # df is the degrees of freedom
tstar = qt(0.5+gamma/2, df)
ME = tstar*s/sqrt(n)
LB = xbar-ME
UB = xbar+ME
cat(sprintf("The lower bound: %.4f\nThe upper bound: %.4f",LB,UB))
