Determine the sample statistics (\(\bar{x}\) and \(s\)). I would recommend using a spreadsheet or R (or another computer-based method). \[\bar{x} = -38.2\] \[s = 133 \] Evaluate the \(t\) statistic.

\[t = \frac{|-38.2-0|}{(133)/\sqrt{60}} = 2.22 \] Restate the \(p\)-value. \[p\text{-value} = P\left(|T|>2.22\right) \] Remember, the degree of freedom is one less than the sample size. \[\text{df} = 59 \]

At this point, there are various ways to determine the \(p\)-value. The least accurate way is to use the \(t\)-table. Go to the row with \(\text{df}=n-1=60-1=59\).


If we go to the row with df=59, we can see that our calculated \(t\), 2.22, is between 2.1 and 2.39. Thus, we know \(P(|T|>2.22)\) is between 0.02 and 0.04. You could get pretty close by using linear interpolation. \[P(|T|>2.22)\approx \left(\frac{2.22-2.1}{2.39-2.1}\right)\left(0.02-0.04\right)+0.04 = 0.0317\]

The more accurate \(p\)-value is 0.0299739. This can be calculated using a computer; for example, you can use this web app (for full accuracy, you’ll need the more precise value of \(t=2.2242076\)). You can also use a spreadsheet or R.
