Determine the sample statistics (\(\bar{x}\) and \(s\)). I would recommend using a spreadsheet or R (or another computer-based method). \[\bar{x} = 40.43\] \[s = 106.7 \] Evaluate the \(t\) statistic.

\[t = \frac{|40.43-0|}{(106.7)/\sqrt{70}} = 3.17 \] Restate the \(p\)-value. \[p\text{-value} = P\left(|T|>3.17\right) \] Remember, the degree of freedom is one less than the sample size. \[\text{df} = 69 \]

At this point, there are various ways to determine the \(p\)-value. The least accurate way is to use the \(t\)-table. Go to the row with \(\text{df}=n-1=70-1=69\).


If we go to the row with df=69, we can see that our calculated \(t\), 3.17, is between 2.98 and 3.21. Thus, we know \(P(|T|>3.17)\) is between 0.002 and 0.004. You could get pretty close by using linear interpolation. \[P(|T|>3.17)\approx \left(\frac{3.17-2.98}{3.21-2.98}\right)\left(0.002-0.004\right)+0.004 = 0.00235\]

The more accurate \(p\)-value is 0.0022792. This can be calculated using a computer; for example, you can use this web app (for full accuracy, you’ll need the more precise value of \(t=3.1692053\)). You can also use a spreadsheet or R.
