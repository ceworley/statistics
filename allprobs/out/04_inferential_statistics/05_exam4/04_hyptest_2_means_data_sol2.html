To do this by hand, you first calculate the sample statistics. \[n_1 = 13\] \[n_2 = 37\] \[\bar{x}_1 = 479.6\] \[\bar{x}_2 = 460.3\] \[s_1 = 27.9\] \[s_2 = 23.3\] \[|t| = 2.23\] \[\text{df} = 18.26\] You will get quite close if you round df down (floor).

Then, using a computer application or a \(t\) table, you should be able to determine the following probabilities (using interpolation to estimate if using \(t\) table). \[P(T<2.23) = 0.9809 \] \[P(|T|>2.23) = 0.0381 \] \[p\text{-value} = 0.0381 \]

The \(p\)-value is less than 0.05, so the result is significant.

read.csv

Download triglyceride.csv and move it to a directory (folder). Make a script, welchttest.r, and put it in the same directory. Set the working directory to this directory. Then, run the script.

### welchttest.r
data = read.csv("triglyceride.csv")
x1 = data$x1
x2 = data$x2
