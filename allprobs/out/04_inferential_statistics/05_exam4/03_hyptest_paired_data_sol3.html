Determine the sample size and degrees of freedom. \[n = 15\] \[\text{df} = 14 \] Determine the sample mean of the differences. \[\bar{d} = \frac{\sum d}{n}= 26.12\] Determine the standard deviation of the differences. You probably do not want to do this by hand. \[s_d = \sqrt{\frac{\sum (d-\bar{d})^2}{n-1}} = 35.02 \] Calculate the \(t\)-score. \[t = \frac{|\overline{d}-0|}{s_d/\sqrt{n}} = \frac{26.12}{35.02\sqrt{15}} = 2.89\] Restate the \(p\)-value. \[p\text{-value} = P\left(|T|>2.89\right) \] I would recommend using a computer program to determine this \(p\)-value, like a spreadsheet or R.

\[p\text{-value} = 0.012 \]

The \(p\)-value is less than 0.05, so the result is significant.

R

Make/use a directory (folder) for this problem (paired-data \(t\) test), and set the working directory accordingly. Save run_times.csv to your working directory. I would recommend saving the following script as paired_data_hypothesis_test.r, in the same directory.

table = read.csv("run_times.csv")
x1 = table[['x1']]
x2 = table[['x2']]
t.test(x1,x2,paired=T)

Or, if you wanted to do it the long way:

table = read.csv("run_times.csv")
x1 = table[["x1"]]
x2 = table[["x2"]]
d = x2-x1
n = length(d)
t = abs(mean(d))/(sd(d)/sqrt(n))
cumulative = pt(t,n-1)
pvalue = 2*(1-cumulative)
