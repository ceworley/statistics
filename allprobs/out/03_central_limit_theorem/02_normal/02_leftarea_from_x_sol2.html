First, draw a sketch. We can label the \(Q\) axis by adding integer multiples of 0.05 to 0.16. We know to shade toward the left because small values of \(Q\) satisfy the condition \(Q<0.11\).


We are given a specific \(q\) value as a boundary. (Remember, for random variables we use uppercase letters, but for specific values we use lowercase.) \[q = 0.11 \] We calculate the \(z\) value of the boundary. \[\begin{aligned} z &= \frac{q-\mu}{\sigma} \\\\ &= \frac{0.11-0.16}{0.05} \\\\ &= -1 \end{aligned}\]

We have rephrased our problem into a standard normal probability problem, because \[P(Q<0.11) ~=~ P(Z<-1) \]

So, we just need to evaluate \(P(Z<-1)\). To do this, you just need a \(z\)-table.

\(z\) \(P(Z<z)\)
-1.02 0.1539
-1.01 0.1562
-1 0.1587
-0.99 0.1611
-0.98 0.1635

Thus, we find our answer. \[P(Q<0.11) = 0.1587 \]
