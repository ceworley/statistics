First, draw a sketch. We can label the \(C\) axis by adding integer multiples of 50 to 420. We know to shade toward the left because small values of \(C\) satisfy the condition \(C<430\).


We are given a specific \(c\) value as a boundary. (Remember, for random variables we use uppercase letters, but for specific values we use lowercase.) \[c = 430 \] We calculate the \(z\) value of the boundary. \[\begin{aligned} z &= \frac{c-\mu}{\sigma} \\\\ &= \frac{430-420}{50} \\\\ &= 0.2 \end{aligned}\]

We have rephrased our problem into a standard normal probability problem, because \[P(C<430) ~=~ P(Z<0.2) \]

So, we just need to evaluate \(P(Z<0.2)\). To do this, you just need a \(z\)-table.

\(z\) \(P(Z<z)\)
0.18 0.5714
0.19 0.5753
0.2 0.5793
0.21 0.5832
0.22 0.5871

Thus, we find our answer. \[P(C<430) = 0.5793 \]
