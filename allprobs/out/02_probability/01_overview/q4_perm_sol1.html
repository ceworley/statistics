This scenario describes a permutations problem (order matters). We are considering the nonrepeating sequences of size 3 from a set of size 18.

\[\begin{aligned} {_nP_r} &= \frac{n!}{(n-r)!} \\\\ n &= 18 \\\\ r &= 3 \\\\ {_{18}P_{3}} &= \frac{18!}{(18-3)! } \\\\ &= \frac{18!}{15!} \\\\ &= 18\cdot17\cdot16 \\\\ &= 4896 \end{aligned}\]
