First, determine \(t^\star\) such that \(P(|T|<t^\star)=0.9\). It can help to draw a sketch.


The entire area under the density curve is 1. Thus, we can determine the areas of the tails (using symmetry). \[\text{area of each tail} = \frac{1-\gamma}{2} = \frac{1-0.9}{2} = 0.05\]


Thus, we determine a left area. \[P(T<t^\star)=\gamma + \frac{1-\gamma}{2}= 0.9+0.05=0.95\] It should be mentioned the expression can be simplified. \[P(T<t^\star)= \frac{1}{2}+\frac{\gamma}{2}= 0.5+0.45=0.95\]


We rephrase the puzzle. We wish to determine \(t^\star\) such that \(P(T<t^\star)=0.95\). This is easy to determine using a \(t\)-table, qt in R, t.inv in spreadsheets, or other methods for evaluating the quantile function.

Because \(n=28\), then \(\text{df}=27\).

\[P(T<1.7)=0.95\] In the original phrasing,

\[P(|T|<1.7)=0.9\]

So, \(t^\star = 1.7\). Now, use the given expressions.

\[\text{Lower bound} = 75.9-(1.7)(22.5)/\sqrt{28} ~=~ 68.7\] \[\text{Upper bound} = 75.9+(1.7)(22.5)/\sqrt{28} ~=~ 83.1\]

Spreadsheet

A spreadsheet can use the confidence.t function. It takes three arguments:

  • Alpha (\(\alpha\)) is the two-tail area, \(\alpha=1-\gamma=1-0.9=0.1\)
  • sample standard deviation, \(s=22.5\)
  • Sample size, \(n=28\)

The function returns the margin of error. \[\text{margin of error} = \frac{t^\star s}{\sqrt{n}}\]

So, in this case, if in a spreadsheet you typed =confidence.t(1-0.9,22.5,28), the result would be the margin of error, 7.2425533. You then add/subtract the margin of error to/from the sample mean. \[\bar{x}-\text{ME} = 75.9-7.243 = 68.657\] \[\bar{x}+\text{ME} = 75.9+7.243 = 83.143\]

You can do this with a spreadsheet.


R

xbar = 75.9
s = 22.5
n = 28
gamma = 0.9
df = n-1   # df is the degrees of freedom
tstar = qt(0.5+gamma/2, df)
ME = tstar*s/sqrt(n)
LB = xbar-ME
UB = xbar+ME
cat(sprintf("The lower bound: %.4f\nThe upper bound: %.4f",LB,UB))
## The lower bound: 68.6574
## The upper bound: 83.1426
