First, determine \(z^\star\) such that \(P(|Z|<z^\star)=0.93\). It can help to draw a sketch.


The entire area under the density curve is 1. Thus, we can determine the areas of the tails (using symmetry). \[\text{area of each tail} = \frac{1-\gamma}{2} = \frac{1-0.93}{2} = 0.035\]


Determine a leftward area with boundary \(z^\star\). \[P(Z<z^\star)=\gamma + \frac{1-\gamma}{2}= 0.93+0.035=0.965\] It should be mentioned the expression can be simplified. \[P(Z<z^\star)= \frac{1}{2}+\frac{\gamma}{2}= 0.5+0.465=0.965\]


We rephrase the puzzle. We wish to determine \(z^\star\) such that \(P(Z<z^\star)=0.965\). This is easy to determine using a \(z\)-table, qnorm in R, norm.inv in spreadsheets, or other methods for evaluating the quantile function.

\[P(Z<1.81)=0.965\] In the original phrasing,

\[P(|Z|<1.81)=0.93\]

So, \(z^\star = 1.81\). Now, use the given expressions.

\[\text{Lower bound} = 57.6-(1.81)(16.3)/\sqrt{147} ~=~ 55.2\] \[\text{Upper bound} = 57.6+(1.81)(16.3)/\sqrt{147} ~=~ 60\]

Spreadsheet

A spreadsheet can use the confidence.norm function. It takes three arguments:

  • Alpha (\(\alpha\)) is the two-tail area, \(\alpha=1-\gamma=1-0.93=0.07\)
  • Population standard deviation, \(\sigma=16.3\)
  • Sample size, \(n=147\)

The function returns the margin of error. \[\text{margin of error} = \frac{z^\star \sigma}{\sqrt{n}}\]

So, in this case, if in a spreadsheet you typed =confidence.norm(1-0.93,16.3,147), the result would be the margin of error, 2.4359351. You then add/subtract the margin of error to/from the sample mean. \[\bar{x}-\text{ME} = 57.6-2.436 = 55.164\] \[\bar{x}+\text{ME} = 57.6+2.436 = 60.036\]

You can do this with a spreadsheet.


R

xbar = 57.6
sigma = 16.3
n = 147
gamma = 0.93
zstar = qnorm(gamma+(1-gamma)/2)
ME = zstar*sigma/sqrt(n)
LB = xbar-ME
UB = xbar+ME
cat(sprintf("The lower bound: %.4f\nThe upper bound: %.4f",LB,UB))
