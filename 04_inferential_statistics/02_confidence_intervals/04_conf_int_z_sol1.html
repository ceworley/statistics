First, determine \(z^\star\) such that \(P(|Z|<z^\star)=0.85\). It can help to draw a sketch.


The entire area under the density curve is 1. Thus, we can determine the areas of the tails (using symmetry). \[\text{area of each tail} = \frac{1-\gamma}{2} = \frac{1-0.85}{2} = 0.075\]


Determine a leftward area with boundary \(z^\star\). \[P(Z<z^\star)=\gamma + \frac{1-\gamma}{2}= 0.85+0.075=0.925\] It should be mentioned the expression can be simplified. \[P(Z<z^\star)= \frac{1}{2}+\frac{\gamma}{2}= 0.5+0.425=0.925\]


We rephrase the puzzle. We wish to determine \(z^\star\) such that \(P(Z<z^\star)=0.925\). This is easy to determine using a \(z\)-table, qnorm in R, norm.inv in spreadsheets, or other methods for evaluating the quantile function.

\[P(Z<1.44)=0.925\] In the original phrasing,

\[P(|Z|<1.44)=0.85\]

So, \(z^\star = 1.44\). Now, use the given expressions.

\[\text{Lower bound} = 75.6-(1.44)(17.9)/\sqrt{252} ~=~ 74\] \[\text{Upper bound} = 75.6+(1.44)(17.9)/\sqrt{252} ~=~ 77.2\]

Spreadsheet

A spreadsheet can use the confidence.norm function. It takes three arguments:

  • Alpha (\(\alpha\)) is the two-tail area, \(\alpha=1-\gamma=1-0.85=0.15\)
  • Population standard deviation, \(\sigma=17.9\)
  • Sample size, \(n=252\)

The function returns the margin of error. \[\text{margin of error} = \frac{z^\star \sigma}{\sqrt{n}}\]

So, in this case, if in a spreadsheet you typed =confidence.norm(1-0.85,17.9,252), the result would be the margin of error, 1.6232071. You then add/subtract the margin of error to/from the sample mean. \[\bar{x}-\text{ME} = 75.6-1.623 = 73.977\] \[\bar{x}+\text{ME} = 75.6+1.623 = 77.223\]

You can do this with a spreadsheet.


R

xbar = 75.6
sigma = 17.9
n = 252
gamma = 0.85
zstar = qnorm(gamma+(1-gamma)/2)
ME = zstar*sigma/sqrt(n)
LB = xbar-ME
UB = xbar+ME
cat(sprintf("The lower bound: %.4f\nThe upper bound: %.4f",LB,UB))
