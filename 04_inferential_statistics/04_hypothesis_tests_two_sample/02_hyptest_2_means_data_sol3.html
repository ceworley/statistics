To do this by hand, you first calculate the sample statistics. \[n_1 = 29\] \[n_2 = 20\] \[\bar{x}_1 = 302.2\] \[\bar{x}_2 = 325.4\] \[s_1 = 24.7\] \[s_2 = 30.2\] \[|t| = 2.83\] \[\text{df} = 35.474\] You will get quite close if you round df down (floor).

Then, using a computer application or a \(t\) table, you should be able to determine the following probabilities (using interpolation to estimate if using \(t\) table). \[P(T<2.83) = 0.9962 \] \[P(|T|>2.83) = 0.0076 \] \[p\text{-value} = 0.0076 \]

read.csv

Download triglyceride.csv and move it to a directory (folder). Make a script, welchttest.r, and put it in the same directory. Set the working directory to this directory. Then, run the script.

### welchttest.r
data = read.csv("triglyceride.csv")
x1 = data$x1
x2 = data$x2
t.test(x1,x2)
