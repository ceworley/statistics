First, determine the interval of typical measurements.

\[\begin{aligned} \text{interval of typical measurements} &= (\mu-2\sigma,\,\mu+2\sigma)\\ &= (51,59) \end{aligned}\]

Now, determine how many measurements (and divide by 200 for what proportion) are either less than 51 or more than 59. You’ll want to use a computer.

R

x = read.csv("check_interval_typical_measurements.csv")$x
n = length(x)
count_outside = sum(x<51 | x>59)
prop_outside = count_outside/n
print(prop_outside)
## [1] 0.03

In R, the “|” operator means “or”.

You could also write the equality as an absolute deviation from the mean. Any measurement more than 4 units from 55 (in either direction) would be outside the interval.

x = read.csv("check_interval_typical_measurements.csv")$x
n = length(x)
count_outside = sum( abs(x-55)>4 )
prop_outside = count_outside/n
print(prop_outside)
## [1] 0.03

Spreadsheet

In a spreadsheet you can use the IF function along with the OR function to determine which measurements are under 51 or over 59. You then use the SUM function to count the 1s.
