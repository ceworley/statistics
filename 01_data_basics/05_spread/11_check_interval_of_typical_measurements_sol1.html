First, determine the interval of typical measurements.

\[\begin{aligned} \text{interval of typical measurements} &= (\mu-2\sigma,\,\mu+2\sigma)\\ &= (10,50) \end{aligned}\]

Now, determine how many measurements (and divide by 200 for what proportion) are either less than 10 or more than 50. You’ll want to use a computer.

R

x = read.csv("check_interval_typical_measurements.csv")$x
n = length(x)
count_outside = sum(x<10 | x>50)
prop_outside = count_outside/n
print(prop_outside)
## [1] 0.04

In R, the “|” operator means “or”.

You could also write the equality as an absolute deviation from the mean. Any measurement more than 20 units from 30 (in either direction) would be outside the interval.

x = read.csv("check_interval_typical_measurements.csv")$x
n = length(x)
count_outside = sum( abs(x-30)>20 )
prop_outside = count_outside/n
print(prop_outside)
## [1] 0.04

Spreadsheet

In a spreadsheet you can use the IF function along with the OR function to determine which measurements are under 10 or over 50. You then use the SUM function to count the 1s.
