First, draw a sketch. We can label the \(D\) axis by adding integer multiples of 50 to 180. We know to shade toward the right because large values of \(D\) satisfy the condition \(D>140\).


We are given a specific \(d\) value as a boundary. (Remember, for random variables we use uppercase letters, but for specific values we use lowercase.) \[d = 140 \] We calculate the \(z\) value of the boundary. \[\begin{aligned} z &= \frac{d-\mu}{\sigma} \\\\ &= \frac{140-180}{50} \\\\ &= -0.8 \end{aligned}\]

We have rephrased our problem into a standard normal probability problem, because \[P(D>140) ~=~ P(Z>-0.8) \]

So, we just need to evaluate \(P(Z>-0.8)\).

To do this, you need to remember that right-area events are complementary to left-area events. \[P(Z>-0.8) ~=~ 1-P(Z<-0.8) \]

You can use a \(z\)-table.

\(z\) \(P(Z<z)\)

Thus, we find our answer. \[\begin{aligned} P(D>140) &= 1-0.2119 \\ &= 0.7881 \end{aligned}\]
